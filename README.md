gaego-init
===============

GAE/Go init tools

## Usage

```
$ gaego-init -a <your app name>
$ cd <your app name>
$ make run
$ curl http://localhost:8080
```

## Installation

```
$ go get -d github.com/kyokomi/gaego-init
$ cd $GOPATH/src/github.com/kyokomi/gaego-init
$ make install  
```
